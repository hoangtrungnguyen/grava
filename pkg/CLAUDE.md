# pkg/

Public Go packages — the bulk of grava's implementation. Each subdirectory is one package with its own `doc.go` and (often) `README.md`. Includes CLI commands (`cmd`), core domain (`core`, `graph`, `coordinator`, `orchestrator`), Dolt integration (`dolt`, `doltinstall`, `migrate`), git plumbing (`gitattributes`, `gitconfig`, `githooks`, `merge`), and shared utilities (`log`, `errors`, `idgen`, `notify`, `utils`, `validation`).
